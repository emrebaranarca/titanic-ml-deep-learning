# Titanic Survival Prediction — ML & Deep Learning

A machine learning and deep learning project that predicts passenger survival on the Titanic. Built for the [Kaggle Titanic Competition](https://www.kaggle.com/competitions/titanic).

| `titanic-survivor-classifier` | ML-focused |

## Overview

This project walks through a complete machine learning pipeline:

1. **Exploratory Data Analysis (EDA)** — visual and statistical analysis of the Titanic dataset
2. **Data Preprocessing** — handling missing values, encoding categorical features
3. **Feature Engineering** — title extraction, family size, age groups, fare categories, cabin indicator
4. **Model Training & Comparison**
   - Logistic Regression
   - Random Forest
   - XGBoost
   - Deep Learning (PyTorch neural network)
5. **Hyperparameter Optimization** — GridSearchCV for XGBoost
6. **Kaggle Submission** — generates `submission.csv`

## Tech Stack
